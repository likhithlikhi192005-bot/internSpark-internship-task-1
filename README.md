# internSpark-internship-task-1

## Python File Automation Script

This project is a Python-based file automation tool that simplifies common file management tasks. It allows users to rename, sort, move, and delete files through an interactive command-line interface.

### Features

* Rename files and folders
* Sort files into folders based on their extensions
* Move files to a specified directory
* Delete unwanted files safely
* User-friendly menu-driven interface
* Uses the **OS module** for file operations
* Implements **exception handling** for reliable error management
* Generates **log files** to record every operation performed
* Accepts **user input** for flexible execution

### Technologies Used

* Python 3
* OS Module
* Logging Module
* Exception Handling
